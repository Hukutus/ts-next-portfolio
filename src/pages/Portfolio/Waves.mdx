**Motivation**

I saw a wave effect on another website, which made me want to try programming them myself.

The wave image on this page is randomly generated on load, so you can reload the page to create a different wave.

**Technical stuff**

The wave is a quite simple **SVG** path with randomized y-points. A **Bezier command** is used to round out the lines going through each point.

**Motivation**

I just wanted to make some waves you can see on many web pages these days. Refresh the page to see a different wave.
As you've probably noticed, there's a wave component inside each of the navigation buttons.

**Technical stuff**

The wave is an **SVG** path with randomized y-points. **Bezier command** is used to round out the lines going through each point.
Lastly the path goes around the bottom to complete the path to fill it with color.
